# Status Codes

> <- Back to [[HTTP]]

Three-digit codes indicating the result of an HTTP request. 2xx success (200 OK, 201 Created, 204 No Content). 3xx redirect (301 Moved Permanently, 302 Found, 304 Not Modified). 4xx client error (400 Bad Request, 401 Unauthorized, 403 Forbidden, 404 Not Found, 429 Too Many Requests). 5xx server error (500 Internal, 502 Bad Gateway, 503 Unavailable, 504 Timeout).
