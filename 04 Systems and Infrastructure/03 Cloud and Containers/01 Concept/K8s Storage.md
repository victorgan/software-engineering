# K8s Storage

> <- Back to [[Container Orchestration]]

Kubernetes storage abstractions for persistent data. PersistentVolumes (PV) represent storage resources. PersistentVolumeClaims (PVC) are requests for storage by pods. StorageClasses enable dynamic provisioning of storage from cloud providers (EBS, GCE PD, Azure Disk).
