# K8s Scaling

> <- Back to [[Container Orchestration]]

Automatic scaling in Kubernetes at multiple levels. HPA (Horizontal Pod Autoscaler) scales the number of pod replicas based on CPU, memory, or custom metrics. VPA (Vertical Pod Autoscaler) adjusts resource requests and limits. Cluster Autoscaler adds or removes nodes based on pending pod demand.
