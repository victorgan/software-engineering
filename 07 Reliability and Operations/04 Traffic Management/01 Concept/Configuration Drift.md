# Configuration Drift

> <- Back to [[Configuration Management in Production]]

The divergence between intended configuration and actual system state over time. Drift occurs when manual changes are made outside of configuration management, creating inconsistencies. Detection and remediation tools ensure systems remain in their desired state.
