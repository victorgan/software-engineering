# Metric Types

> <- Back to [[Metrics]]

The fundamental types of numeric measurements: Counter (monotonically increasing), Gauge (current value that can go up or down), Histogram (distribution of values), and Summary (pre-calculated quantiles). Each type serves different monitoring needs.
