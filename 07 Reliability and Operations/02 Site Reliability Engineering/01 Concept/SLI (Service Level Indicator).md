# SLI (Service Level Indicator)

> <- Back to [[SLAs, SLOs, SLIs]]

A measurable metric that quantifies service behavior from the user's perspective. Examples include latency p99, availability percentage, and error rate. SLIs should be directly tied to user experience, not internal system metrics.
