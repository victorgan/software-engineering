# Firewalls

> <- Back to [[Network Security]]

Network-level and application-level traffic filtering. Network firewalls (security groups, NACLs) filter by IP, port, and protocol. Web Application Firewalls (WAF) inspect HTTP traffic for malicious patterns like SQL injection and XSS.
