# HMAC

> <- Back to [[Hashing]]

Hash-based Message Authentication Code. HMAC combines a hash function with a secret key to verify both data integrity and authenticity. Unlike a plain hash, HMAC ensures the message was not tampered with AND was produced by someone with the secret key.
