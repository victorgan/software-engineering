# TLS 1.3

> <- Back to [[TLS and SSL]]

The latest version of TLS, offering significant improvements over TLS 1.2: reduced handshake latency (1-RTT, optional 0-RTT), removal of weak cipher suites (RC4, 3DES, static RSA), mandatory forward secrecy, and a simplified, more secure protocol design.
