# Penetration Testing

**Back:** [[Security Operations]]

Authorized simulated attacks against systems to identify exploitable vulnerabilities. Penetration testing ranges from black box (no prior knowledge) to white box (full access) and follows a methodology of reconnaissance, scanning, exploitation, and reporting.
