# Penetration Testing Types

> <- Back to [[Penetration Testing]]

Three approaches based on information provided to testers: Black box (no prior knowledge, simulates external attacker), White box (full access to source code and architecture, thorough coverage), and Gray box (partial knowledge, balances realism with depth).
