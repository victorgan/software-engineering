# Threat Modeling

**Back:** [[Application Security]]

A structured approach to identifying security threats, vulnerabilities, and mitigations during the design phase. Threat modeling uses frameworks like STRIDE, attack trees, and data flow diagrams to systematically analyze what can go wrong and prioritize security investments.
