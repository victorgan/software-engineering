# SLSA Framework

> <- Back to [[Software Supply Chain]]

Supply-chain Levels for Software Artifacts, a framework for ensuring build integrity. SLSA defines four levels of increasing assurance, from basic build provenance (Level 1) to hermetic, reproducible builds with provenance attestation (Level 4).
