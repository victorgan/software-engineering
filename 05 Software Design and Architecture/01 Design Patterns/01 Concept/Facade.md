# Facade

> <- Back to [[Structural Patterns]]

Simplified interface to a complex subsystem. Provides a unified, higher-level interface that makes the subsystem easier to use. Does not prevent direct access to subsystem components when needed.

## Key Properties

- [[Subsystem Simplification]]
- [[Loose Coupling to Subsystem]]
- [[Entry Point]]

- [[Mediator]] (centralizes communication between peers)
