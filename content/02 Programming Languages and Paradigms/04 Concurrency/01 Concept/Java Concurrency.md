# Java Concurrency

> <- Back to [[Language-Specific Concurrency]]

Java provides a rich concurrency toolkit in `java.util.concurrent` including executors, concurrent collections, locks, and atomic variables. Virtual threads (Project Loom) bring lightweight threading. `synchronized` and `volatile` provide built-in synchronization primitives.
