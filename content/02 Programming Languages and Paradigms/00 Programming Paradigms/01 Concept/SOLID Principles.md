# SOLID Principles

> <- Back to [[Object-Oriented Programming (OOP)]]

Five design principles for writing maintainable, flexible object-oriented code. SOLID stands for Single Responsibility, Open/Closed, Liskov Substitution, Interface Segregation, and Dependency Inversion.
