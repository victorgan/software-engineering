# Zero-Downtime Deployments

> <- Back to [[Deployment Strategies]]

Deploying without any period of unavailability. Achieved through connection draining, graceful shutdown, readiness gates, and pre-stop hooks. Requires applications to handle in-flight requests during shutdown.
