# Drift Detection (CI-CD)

> <- Back to [[GitOps Deployment]]

Continuously comparing actual state to desired state and auto-remediating differences. Drift occurs when manual changes are made outside Git. GitOps controllers detect and correct drift, ensuring the running system matches the Git-declared state.
