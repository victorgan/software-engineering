# MapReduce (Internal)

> <- Back to [[Google Data Processing]]

Original distributed data processing framework (2004 paper). Map phase processes input in parallel; reduce phase aggregates results. Externalized as Hadoop MapReduce. Superseded internally by FlumeJava/Flume. Paper: "MapReduce: Simplified Data Processing on Large Clusters" (2004).
