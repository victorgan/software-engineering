# Dapper

> <- Back to [[Google Monitoring and Operations]]

Google's distributed tracing system (2010 paper). Traces requests across microservices with low overhead. Inspired Zipkin, Jaeger, and OpenTelemetry. Externalized as Cloud Trace (GCP). Paper: "Dapper, a Large-Scale Distributed Systems Tracing Infrastructure" (2010).
