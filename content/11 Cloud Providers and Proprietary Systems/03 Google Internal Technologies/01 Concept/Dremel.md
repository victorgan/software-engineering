# Dremel

> <- Back to [[Google Data Processing]]

Interactive query engine for nested data (2010 paper). Columnar storage with tree-shaped distributed execution. Externalized as BigQuery (GCP). Inspired Apache Drill, Presto/Trino, and Apache Impala. Paper: "Dremel: Interactive Analysis of Web-Scale Datasets" (2010).
