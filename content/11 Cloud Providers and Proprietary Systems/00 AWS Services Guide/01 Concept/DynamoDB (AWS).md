# DynamoDB (AWS)

> <- Back to [[AWS Databases]]

Managed NoSQL key-value and document database. Single-digit millisecond latency at any scale. Auto-scaling throughput, global tables for multi-region replication, DynamoDB Streams for change data capture. Pay-per-request or provisioned capacity.
