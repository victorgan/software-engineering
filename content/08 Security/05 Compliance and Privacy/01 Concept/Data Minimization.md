# Data Minimization

> <- Back to [[Privacy Engineering]]

Collecting only the data that is strictly necessary for the stated purpose and retaining it only as long as needed. Data minimization reduces risk (you cannot breach data you do not have), simplifies compliance, and builds user trust.
