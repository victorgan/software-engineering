# PCI-DSS

> <- Back to [[Regulatory Frameworks]]

Payment Card Industry Data Security Standard. Defines requirements for handling payment card data: network segmentation, encryption, access control, vulnerability management, and regular security assessments. Non-compliance can result in fines and loss of payment processing ability.
