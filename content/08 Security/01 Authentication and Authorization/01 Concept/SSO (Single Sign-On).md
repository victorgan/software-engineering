# SSO (Single Sign-On)

> <- Back to [[Authentication Methods]]

A centralized authentication mechanism where one login grants access to multiple services. SSO improves user experience (fewer passwords), security (centralized credential management), and IT operations (centralized provisioning and deprovisioning).
