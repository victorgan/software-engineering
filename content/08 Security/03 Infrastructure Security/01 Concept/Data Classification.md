# Data Classification

> <- Back to [[Data Protection]]

Categorizing data by sensitivity level: Public (no restrictions), Internal (business use), Confidential (restricted access), and Restricted (highest protection, PII/PHI/financial). Classification drives access controls, encryption requirements, and retention policies.
