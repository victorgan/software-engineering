# Secrets Management Tools

> <- Back to [[Secrets Management]]

Dedicated tools for secret storage and distribution: HashiCorp Vault (full-featured, self-hosted), AWS Secrets Manager (managed, integrated), Azure Key Vault (managed, integrated), and SOPS (encrypted files in git). These tools provide audit logging, access control, and rotation capabilities.
