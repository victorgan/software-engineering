# Key Management

> <- Back to [[PKI (Public Key Infrastructure)]]

The practices and systems for generating, storing, distributing, rotating, and destroying cryptographic keys. Hardware Security Modules (HSMs) provide tamper-resistant key storage. Key rotation limits the exposure window of compromised keys.
