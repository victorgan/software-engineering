# Supply Chain Security

**Back:** [[08 - Security MOC]]

Supply chain security protects the software build and delivery pipeline from compromise. From dependency vulnerabilities and artifact signing to CI/CD hardening, supply chain security ensures that what you build and deploy has not been tampered with and does not contain known vulnerabilities.
