# Pipeline Hardening

> <- Back to [[CI-CD Security]]

Securing CI/CD pipelines by applying least privilege, using no long-lived secrets, running ephemeral build agents, isolating build environments, and auditing pipeline configurations. CI/CD systems are high-value targets because they have access to production deployment credentials.
