# Patch Management

> <- Back to [[Vulnerability Management]]

The process of applying security patches prioritized by CVSS severity within defined SLAs. Critical patches should be applied within hours to days; high-severity within days to weeks. Automated patching reduces the window of exposure for known vulnerabilities.
