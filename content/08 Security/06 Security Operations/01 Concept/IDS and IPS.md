# IDS and IPS

> <- Back to [[Security Monitoring]]

Intrusion Detection Systems (IDS) monitor and alert on suspicious activity; Intrusion Prevention Systems (IPS) can also block detected threats. Both can be network-based (monitoring traffic) or host-based (monitoring system activity).
