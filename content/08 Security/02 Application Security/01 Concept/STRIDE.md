# STRIDE

> <- Back to [[Threat Modeling]]

A threat classification framework: Spoofing (identity), Tampering (data modification), Repudiation (denying actions), Information Disclosure (data leaks), Denial of Service (availability), and Elevation of Privilege (unauthorized access). STRIDE systematically identifies threats for each component in a system.
