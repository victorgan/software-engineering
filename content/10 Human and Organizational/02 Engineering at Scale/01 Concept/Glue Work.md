# Glue Work

> <- Back to [[Technical Leadership]]

Essential non-promotable work: documentation, onboarding, coordination, project management, code reviews, improving developer experience. Tanya Reilly's concept. This work must be recognized, valued, and shared equitably -- not disproportionately borne by underrepresented groups.
