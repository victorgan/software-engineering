# Accessibility

**Back:** [[12 - Web and Mobile Development MOC]]

Making web applications usable by everyone, including people with disabilities. WCAG provides the guidelines, ARIA fills gaps in semantic HTML, and screen readers are the primary assistive technology for vision-impaired users. Good accessibility often improves usability for all users.

## Categories

- WCAG Guidelines
- ARIA Roles
- Semantic HTML
- Screen Readers
- Keyboard Navigation
- Color Contrast
- Focus Management
