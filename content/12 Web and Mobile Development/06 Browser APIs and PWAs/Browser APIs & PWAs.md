# Browser APIs & PWAs

**Back:** [[12 - Web and Mobile Development MOC]]

Advanced browser capabilities that enable app-like experiences on the web. Service Workers provide offline support, the Cache API enables programmatic caching, and Web Push delivers notifications. Together they form the foundation of Progressive Web Apps.
