# Mobile Development

**Back:** [[12 - Web and Mobile Development MOC]]

Building native and cross-platform mobile applications. iOS development uses Swift and SwiftUI, Android uses Kotlin and Jetpack Compose, while React Native and Flutter enable cross-platform development from a single codebase with different tradeoffs.

## Categories

- iOS (Swift, SwiftUI, UIKit)
- Android (Kotlin, Jetpack Compose)
- Cross-Platform (React Native, Flutter, Kotlin Multiplatform)
- Mobile Lifecycle
- Push Notifications (APNs, FCM)
- Deep Linking
- Offline Storage
- App Distribution
