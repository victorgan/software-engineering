# Editors & IDEs

**Back:** [[13 - Developer Tooling and Productivity MOC]]

The primary tools developers use to write code. VS Code dominates with its extension ecosystem, JetBrains offers deep language-specific intelligence, Vim/Neovim provides unmatched editing efficiency, and LSP has standardized language support across all editors.

## Categories

- VS Code
- JetBrains (IntelliJ, PyCharm, WebStorm, GoLand)
- Vim / Neovim
- Emacs
- LSP (Language Server Protocol)
- DAP (Debug Adapter Protocol)
