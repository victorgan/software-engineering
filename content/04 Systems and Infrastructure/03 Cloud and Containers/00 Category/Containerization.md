# Containerization

**Back:** [[Cloud and Containers]]

Packaging applications and their dependencies into lightweight, portable containers that run consistently across environments. Containers use OS-level virtualization (namespaces, cgroups) rather than hardware virtualization, making them faster and more efficient than VMs.
