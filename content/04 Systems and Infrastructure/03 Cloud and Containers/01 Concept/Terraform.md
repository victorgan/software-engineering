# Terraform

> <- Back to [[Infrastructure as Code]]

HashiCorp's declarative IaC tool. Uses HCL (HashiCorp Configuration Language) to define infrastructure across multiple cloud providers. State management tracks the mapping between config and real resources. Modules enable reusable, composable infrastructure components. Plan before apply for safe changes.
