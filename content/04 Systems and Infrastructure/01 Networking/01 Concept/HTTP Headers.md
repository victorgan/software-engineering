# HTTP Headers

> <- Back to [[HTTP]]

Key-value pairs in HTTP requests and responses that carry metadata. Content-Type specifies the media type, Authorization carries credentials, Cache-Control directs caching behavior, and CORS headers (Access-Control-Allow-Origin, etc.) control cross-origin access.
