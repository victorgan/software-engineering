# Onion Architecture

> <- Back to [[Layered Architecture]]

Similar to hexagonal architecture with concentric layers of dependency. Inner layers define interfaces; outer layers implement them. The core (domain model) has no outward dependencies, surrounded by domain services, application services, and infrastructure.
