# Dynamic Configuration

> <- Back to [[Configuration Management in Production]]

Runtime tuning of system parameters without restarts or deployments using config servers like Consul, etcd, or similar. Dynamic configuration enables operational control for timeouts, rate limits, and other parameters that may need adjustment in real-time.
