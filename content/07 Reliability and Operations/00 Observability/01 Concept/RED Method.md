# RED Method

> <- Back to [[Metrics]]

A monitoring methodology for request-driven services that focuses on three key metrics: Rate (requests per second), Errors (failed requests per second), and Duration (latency distribution). Provides a simple, effective framework for understanding service health from the user's perspective.
