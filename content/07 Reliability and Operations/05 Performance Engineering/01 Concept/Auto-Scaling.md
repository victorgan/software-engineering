# Auto-Scaling

> <- Back to [[Capacity Planning]]

Automatically adjusting resource capacity based on demand. Horizontal scaling adds or removes instances; vertical scaling adjusts instance size. Auto-scaling policies use metrics like CPU utilization, request rate, or queue depth to trigger scaling actions.
