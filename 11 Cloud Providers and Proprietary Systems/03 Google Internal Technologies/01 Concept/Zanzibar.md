# Zanzibar

> <- Back to [[Google Security and Identity]]

Google's global authorization system. Consistent, scalable relationship-based access control (ReBAC). Checks billions of authorization decisions per second. Inspired SpiceDB, Authzed, OpenFGA, and Ory Keto. Paper: "Zanzibar: Google's Consistent, Global Authorization System" (2019).
