# Spanner (Internal)

> <- Back to [[Google Databases and Data]]

Globally distributed, strongly consistent relational database with TrueTime (atomic clocks + GPS for global consistency). Externalized as Cloud Spanner (GCP). Inspired CockroachDB and YugabyteDB. Paper: "Spanner: Google's Globally-Distributed Database" (2012).
