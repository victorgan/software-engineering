# BeyondCorp Paper (2014)

> <- Back to [[Key Papers to Read]]

**"BeyondCorp: A New Approach to Enterprise Security" (2014)** -- Ward, Beyer

Describes Google's shift from perimeter-based security to zero-trust. Key principles: the network is not trusted, access depends on user and device credentials, all access is authenticated/authorized/encrypted. Launched the industry-wide zero-trust movement. Multiple follow-up papers detail the implementation.
