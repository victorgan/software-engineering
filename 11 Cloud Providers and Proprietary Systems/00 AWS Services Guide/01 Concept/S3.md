> <- Back to [[AWS Storage]]

Simple Storage Service. Object storage with buckets, versioning, lifecycle policies, and storage classes (Standard, Intelligent-Tiering, Infrequent Access, Glacier). The foundation for data lakes, static hosting, backups, and log storage. Virtually unlimited capacity.
