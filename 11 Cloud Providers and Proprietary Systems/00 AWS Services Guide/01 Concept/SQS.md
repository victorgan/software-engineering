# SQS

> <- Back to [[AWS Messaging and Events]]

Simple Queue Service. Managed message queue. Standard queues (best-effort ordering, at-least-once delivery) and FIFO queues (exactly-once processing, strict ordering). Used for decoupling services and buffering workloads.
