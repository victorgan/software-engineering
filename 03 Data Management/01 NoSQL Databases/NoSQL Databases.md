# NoSQL Databases

**Back:** [[03 - Data Management MOC]]

Non-relational databases designed for specific data models and access patterns that relational databases handle less efficiently. NoSQL databases trade some relational features (joins, ACID across tables) for scalability, flexibility, or performance in targeted use cases.
