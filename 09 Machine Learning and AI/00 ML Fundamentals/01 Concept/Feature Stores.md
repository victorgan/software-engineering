# Feature Stores

> <- Back to [[Feature Engineering]]

Centralized repositories for storing, managing, and serving ML features. Ensure consistency between training and inference, and enable feature reuse across teams.

## Key Tools

- **Feast** -- open-source feature store
- **Tecton** -- managed feature platform
- **Hopsworks** -- open-source ML platform with feature store
