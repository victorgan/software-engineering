# Monorepo

> <- Back to [[Monorepo vs Polyrepo]]

All code in one repository. Used by Google, Meta, and many large organizations. Enables atomic cross-project changes, shared tooling, and easy code discovery. Requires specialized tooling at scale for build performance.
