# End-to-End Tests

> <- Back to [[Testing Pyramid]]

Test full user flows through the complete system. Slow, expensive, and fewer in number. Provide the highest confidence that the system works from the user's perspective, but are more brittle and harder to debug.

- [[Full System Coverage]]
